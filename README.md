# Ghana Centralized Energy Data Management System

Relational database design for a centralized energy data management system in Ghana — UGBS Project

## Project Overview
A fully normalized relational database designed to consolidate 
operational data across Ghana's electricity value chain, covering 
generation, distribution, billing, payments, outages, and maintenance.
